Installeer flask op vm door pip install flask

Vervolgens maak ik een directory met mkdir my_flask_api

Ga de directory in met cd my_flask_api

Vervolgens doe ik nano app.py om python script te maken

Vervolgens run ik pip3 install mysql-connector-python om te kunnen connecte met mysql.

Nu voeg ik code toe aan mijn app.py script

Nu kun je python3 app.py cmd doen om de api aan te zetten.

Overigens zorg ervoor dat je sudo ufw allow 5000 runned, zodat je met de api kan communiceren anders ga je errors krijgen.
